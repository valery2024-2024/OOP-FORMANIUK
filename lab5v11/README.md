# Лабораторна робота №5 — Generics, колекції, LINQ, винятки (варіант 11)

**Тема:** Узагальнені типи, колекції і LINQ, обробка винятків.  
**Мета:** Навчитися створювати узагальнені класи/методи, працювати з колекціями та LINQ, валідовувати дані і створювати/обробляти винятки.

## Предметна область
**Прайс-лист** (`PriceItem`) і **Кошик** (`CartItem`).  
Елементи кошика зберігають *код товару* і кількість — це **агрегація**, бо посилаються на прайс.

## Структура
- `Repository<T>` — узагальнений репозиторій з операціями **Add/Remove/Find/All/Where**.
- `NotFoundException` — власний виняток, кидається при відсутності товару.
- `CartService` — обчислення:
  - **Проміжна сума** (subtotal)
  - **Знижка купона** (напр., 7%)
  - **Середня ціна за позицію**
  - Запити LINQ (товари дорожчі за threshold)
